API Documentation¶

multiviewica:

permica(X[, n_components, …])

Performs one ICA per group (ex: subject) and align sources using the hungarian algorithm.

groupica(X[, n_components, …])

Performs PCA on concatenated data across groups (ex: subjects) and apply ICA on reduced data.

multiviewica(X[, n_components, …])

Performs MultiViewICA.
